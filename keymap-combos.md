# Keymap Combos & Macros Reference

## Combos

| Key Combo | Output |
|-----------|--------|
| D + F | Tab |
| J + K | Escape |
| F + G | Return |
| A + S | Select All (Cmd+A) |
| S + D | Save (Cmd+S) |
| Z + X | Undo (Cmd+Z) |
| X + C | Copy (Cmd+C) |
| C + V | Paste (Cmd+V) |
| U + I | `../` |
| , + . | Delete Word Back (Alt+Bksp) |
| . + / | Delete Word Forward (Alt+Del) |
| U + J (vertical) | `()` with cursor inside |
| I + K (vertical) | `[]` with cursor inside |
| O + L (vertical) | `{}` with cursor inside |
| K + L | `\|>` |
| L + ; | `=>` |
| ; + ' | `->` |
| 7 + 8 | `!=` |
| 8 + 9 | `===` |
| T + G | `the ` |
| 1 + 2 | Screenshot to Clipboard |
| LShift + RShift | Caps Word |
| Esc + ` | Studio Unlock |
| Lower + Space thumb | Tab Forward (window switch) |
| Raise + Enter thumb | Tab Back (window switch) |
| Lower + G | Browser Tab Left |
| Raise + H | Browser Tab Right |

## Macros

| Macro | Output |
|-------|--------|
| updir | `../` |
| paren_pair | `()` + cursor left |
| bracket_pair | `[]` + cursor left |
| brace_pair | `{}` + cursor left |
| fat_arrow | `=>` |
| thin_arrow | `->` |
| pipe_op | `\|>` |
| not_equal | `!=` |
| triple_eq | `===` |
| m_the | `the ` |
